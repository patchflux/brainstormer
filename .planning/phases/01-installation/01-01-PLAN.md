1. Sets install target: INSTALL_DIR="${HOME}/.claude/brainstomer"

2. Detects script location (handles both direct run and curl pipe):
   - If run directly: use script's directory as source
   - If piped from curl: clone to temp dir, use that as source

3. Creates target directory structure:
   - mkdir -p "$INSTALL_DIR"

4. Copies all required directories preserving structure:
   - agents/ (3 files)
   - workflows/ (3 files)
   - commands/brainstorm/ (4 files)
   - references/ (1 file)
   - templates/ (9 files including templates/research/)

5. Validates installation:
   - Check each required file exists in target
   - Count files copied
   - Exit 1 if any missing

6. Prints success message with:

   - File count
   - Usage instructions showing all 4 commands

Make script idempotent (safe to run multiple times).
Use set -e for fail-fast behavior.
No dependencies beyond core POSIX utilities (sh, mkdir, cp, rm).

For curl install: detect if stdin is not a tty (piped), then git clone to temp, run install from there, cleanup.
